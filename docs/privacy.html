Ai Caption 隱私政策

最後更新：2026 年 6 月 v1.0

簡而言之：Ai Caption 在你的裝置上處理一切。你的影片、音訊、字幕和歌詞永遠不會離開你的電腦。沒有帳號，沒有分析，沒有 Cookie，沒有追蹤——什麼都沒有。

1. 我們不收集的資料 1. Data We Do Not Collect

Ai Caption 的設計核心就是「本地優先」。以下這些東西，我們根本沒有技術手段去取得，因為 App 從不與任何遠端伺服器溝通：

  • 影片、音訊檔案或任何其衍生資料（人聲、波形、頻譜）
  • 字幕、歌詞文字或你貼入的任何參考內容
  • 個人識別資訊（姓名、Email、IP 位址）
  • 使用統計、崩潰報告或診斷資料
  • 裝置識別碼或硬體資訊
  • Cookie、LocalStorage 或任何網頁追蹤機制

2. 唯一的網路活動 2. The One Network Activity

Ai Caption 有且僅有一種主動的網路連線：當你選擇下載 AI 模型時。

  • Demucs 模型（~320 MB）：來自 Hugging Face（huggingface.co）
  • faster-whisper large-v3（~3.1 GB）：來自 Hugging Face
  • torchaudio MMS_FA 對齊模型（~1.2 GB）：來自 Hugging Face
  • Python 依賴項：來自 PyPI（pypi.org）和 pytorch.org

這些下載完全由你主動觸發（在設定中手動點擊），不是自動在背景執行的。下載完成後，Ai Caption 在日常使用時不需要也不會進行任何網路連線。模型下載請求受 Hugging Face 和 PyPI 各自的隱私政策規範——我們對此無法控制，但我們也沒有在這些請求中夾帶任何額外的使用資料。

3. 資料儲存於何處 3. Where Your Data Lives

Ai Caption 的所有資料——影片/音訊檔案、字幕與歌詞結果、匯出文件、App 設定——都儲存在你的本地磁碟上。預設位置為系統的應用程式資料資料夾（Windows: %APPDATA%\Ai Caption），你可以在 App 設定中查看確切路徑，也可以自行移動或刪除這些檔案。

我們沒有雲端同步，沒有備份服務，也沒有任何遠端儲存。你的資料完全在你的掌控之中。

4. 無帳號、無追蹤 4. No Account, No Tracking

使用 Ai Caption 不需要建立帳號、登入或驗證身分。App 內沒有任何分析 SDK（Google Analytics、Firebase、Sentry 等）、廣告網路或第三方追蹤器。

這個網站（行銷頁面）同樣不使用 Cookie 或任何持久性追蹤機制。語言偏好設定（中/英）僅存於你的瀏覽器 localStorage，從不傳送至任何伺服器。

5. 開源透明度 5. Open Source Transparency

Ai Caption 以 MIT 授權條款完全開放原始碼，發布於 GitHub。 任何人都可以審閱程式碼，驗證本政策所述的隱私保護承諾。如果你發現任何與本政策不符的行為，請在 GitHub Issues 提出——我們會認真處理。

6. 未成年人 6. Children

由於 Ai Caption 根本不收集任何個人資料，本 App 對各年齡層使用者均安全。

7. 政策修訂 7. Policy Changes

若本政策有任何重大變更（例如未來版本新增了任何網路功能），我們會在 GitHub Releases 中明確說明，並更新本頁面頂部的「最後更新」日期。

聯絡我們 · Contact

如有任何隱私相關問題，請透過 GitHub Issues 聯繫我們。我們不提供 Email 聯絡方式，以避免垃圾郵件。
