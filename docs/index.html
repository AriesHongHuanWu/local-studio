為什麼選它

準確、私密、隨時可用。

不需要帳號，不需要訂閱，不需要網路——開啟就能用。

免獨立顯卡・筆電也能跑

預設自動挑選適合 CPU 的 int8 模型，Intel Core Ultra / Ryzen 筆電也能順跑。偵測到 GPU 時自動使用，只會更快。

免裝 Python・開箱即用

內建可攜式 Python，不必自己安裝任何執行環境。首次啟動會自動把一切準備就緒。

100% 本地・完全私密

所有處理都在裝置上完成，不上傳任何檔案、無遙測、不連雲端。你的影片與歌詞永遠留在你的電腦。

自我修復・自動補齊

每次啟動都會檢查自己的引擎與模型是否完整，只重新抓回缺少的部分——不必重裝整包。

儲存空間管理

分層查看磁碟用量與剩餘空間：刪除單一模型、清空所有模型，或完整重設——把每一 GB 都看得清清楚楚。

App 內自動更新

新版本一鍵更新，並附上版本說明。介面支援中文 / English，Windows / macOS（Apple Silicon）/ Linux 皆可。

運作原理

四步流水線，一氣呵成。

這是歌詞模式的完整管道——從音訊到逐字時間軸，全程本地推理。模型一次下載，永久離線使用。（影片轉字幕只走語音辨識，更快；清字模式用 LaMa 逐格修補。）

① 人聲分離
Demucs

深度學習模型將人聲從混音中分離，為後續辨識提供乾淨的聲源。

② 語音辨識
faster-whisper

CTranslate2 加速的 Whisper large-v3，逐字時間戳，支援多語。

③ 對齊 / 偏置
強制對齊 · 偏置

完整歌詞→強制對齊（近完美）；部分歌詞/風格→Whisper biasing。

④ 匯出
LRC · SRT · ASS · JSON

生成標準格式文件，可在編輯器中 QA 後再存檔。

自動
自動辨識

無需任何參考——直接轉寫，適合快速草稿或沒有歌詞的歌曲。

歌詞偏置

為誰打造

不論你做影片還是音樂，都適合。

從影片字幕到逐字歌詞，Local Studio 讓字幕與歌詞不再是門檻。

音樂製作人・詞曲創作者

把自己的 demo 或 rough mix 直接拿來對齊歌詞，精確到每個字，不再靠耳朵逐字計時。

歌詞影片・卡拉 OK 製作者

直接匯出 ASS 逐字 \k 掃光格式，丟進 Aegisub 或 After Effects 零人工對時——省去數小時手動作業。

影片剪輯師・字幕製作者

SRT 格式直接導入 Premiere、Final Cut、DaVinci——逐字時間點讓歌詞字幕不再需要手動校對。

語言學習者

跟著精準時間軸學唱日文歌、英文歌——每個字都有起止時間，配合 LRC 播放器逐字跟唱。

引擎內幕

真正的工程，不是噱頭。

每一層都選用業界最佳開源模型，組合成一條完整的本地推理管道。

Demucs htdemucs

Meta AI 開源的深度學習人聲分離模型，精準隔離人聲與伴奏，大幅降低背景噪訊對辨識的干擾。

faster-whisper large-v3

以 CTranslate2 最佳化的 Whisper，在相同硬體上比原版快 4 倍，支援 99 種語言逐字時間戳。

torchaudio MMS_FA + forced_align

基於 torchaudio MMS_FA 的原生 CTC 強制對齊（forced_align），以 uroman 做音素化、免 C++ 編譯器——在有完整歌詞時達到幾近完美的逐字精度。

PyTorch · CUDA cu128

支援 NVIDIA sm_120 架構（RTX 5000 系列），cu128 編譯確保最新 GPU 能充分發揮效能。

React + Tauri v2

輕量 Rust 框架打包 React 介面，執行檔小於 10 MB，不捆綁 Electron 的龐大執行環境。

FastAPI · Python 3.10–3.12

本地 IPC 伺服器橋接 Tauri 前端與 Python 推理後端，跨平台（Windows / macOS / Linux）。

隱私承諾

你的歌曲，只在你的電腦上。

這不是行銷術語——這是設計的核心。

  • 不收集音訊或歌詞資料
  • 不上傳個人資料
  • 無帳號・無分析・無 Cookie
  • 不需要網路連線即可使用
  • 沒有第三方追蹤器
  • 完全在裝置本地執行
唯一的網路活動：當你選擇下載 AI 模型時，程式會從官方來源（Hugging Face、pytorch.org）取得模型檔案。除此之外，Local Studio 不向任何伺服器傳送任何使用資料。
