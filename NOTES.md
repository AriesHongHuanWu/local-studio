# Local Studio v0.1.38

🧰 **Toolbox layout fix** — when you pick a tool, the file drop and that tool's controls now sit at the **top**, with the tool-switch buttons below, so you no longer have to scroll down to run a tool.

Also: the in-app update dialog now shows the **real release notes** (previously it showed a generic placeholder — the updater was reading a hardcoded string instead of this changelog).

Everything from v0.1.37 is included:
- **Mastering** that actually improves the track + 21 genre / hip-hop / artist-style presets, save your own presets.
- New **Downloader + song analyzer** (key / BPM / structure / EQ / vocal-mix advice).

> The Downloader + song analysis need the local engine to include `yt-dlp`. If the **Download** mode shows an "engine missing" notice, run Settings → repair/reinstall engine once. Mastering and the Toolbox work immediately.
